# Gold Reply 02: bugfix 再依頼（前回の報告書あり）

## 使う場面
- 再度の依頼
- 前回の報告書や過去の調査結果がある
- まず受領と進め方を短く示したい

## 返信例
ご相談ありがとうございます。
再度のご依頼でも対応できます。前回の報告書があれば、どこまで確認済みかを把握しやすいので助かります。

この内容なら、15,000円で進められます。
ご購入後に、前回の報告書と今の症状を見比べながら、同じ箇所で止まっているのか、別の原因が混ざっていないかを確認して、原因切り分けから修正まで進めます。別の原因が見つかった場合は、進める前にご相談します。

この内容でよければ、見積もり提案をお送りします。

## 良い理由
- 再依頼を歓迎しつつ、スコープ事故の境界も残している
- 料金案内が短い
- 次アクションが1行で明確

# Gold Reply 03: bugfix 別原因判定

## 使う場面
- 同じ不具合に見えるが、実際には別原因
- 追加料金や別見積りへ切り替える必要がある
- 断ち切らずに、境界を明確に伝えたい

## 返信例
確認結果をご報告します。
今回の症状は、前回の不具合とは別原因と判断しています。

このため、基本料金内でそのまま進めるのではなく、追加修正1件（15,000円）でのご案内になります。
進めるかどうかは、内容を整理したうえで事前にご相談します。追加料金は、合意がないまま発生しません。

必要であれば、この2案の違いも短く整理してお送りします。

## 良い理由
- `範囲外です` で終わらない
- 別原因を明確にしつつ、押しつけ感がない
- 追加料金の不安を先回りして消している

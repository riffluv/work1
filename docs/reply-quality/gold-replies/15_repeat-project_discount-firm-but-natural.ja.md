# Gold Reply 15: repeat project discount firm but natural

## 使う場面
- 以前の対応実績を根拠に、別プロジェクトや別件で値引きを求められる
- buyer が `1万円くらいで` のように具体的な金額を出している
- 講義調にせず、正規の入口へ戻したい

## 返信例

```text
ご連絡ありがとうございます。
別プロジェクトであれば、新しい依頼になります。前回の原因が参考になることはありますが、それだけで10,000円に下げることはしていません。
今回の症状が分かる画面か文言を送ってもらえれば、このまま15,000円で対応できる内容かを返します。
```

## 良い理由
- buyer が出した金額語をそのまま拾って、事実で断れる
- `値引きはしません` の講義で終わらず、次の入口を残せる
- `ご案内します` のような空語に逃げず、何を返すかが見える

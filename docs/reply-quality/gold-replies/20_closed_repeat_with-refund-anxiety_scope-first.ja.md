# Gold Reply 20 — 再発 + 返金不安に、無料保証を約束せず scope を先に返す

## 使いどころ

- closed 後の再相談
- `また似た不具合が出た`
- `返金してほしいと言われている`
- `前回の範囲で無料で見てもらえるか`

## 返信文

```text
ご連絡ありがとうございます。返金の話まで出ると不安になるのは当然だと思います。
今回もまずは、前回の修正とつながっている再発かを確認します。その結果、前回の続きとして対応できるか、新しい依頼になるかを先に返します。返金操作そのものは Stripe 管理画面での対応になるので、必要ならその画面を見ながら案内できます。
似た不具合が出ている画面か文言を送ってください。
```

## この型で守っていること

- 不安を1文で受ける
- 無料保証を即答しない
- `前回範囲か新規か` を先に返す
- 返金操作と不具合確認を混同しない

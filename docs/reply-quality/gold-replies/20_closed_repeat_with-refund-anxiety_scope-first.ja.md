# Gold Reply 20 — 再発 + 返金不安に、無料保証を約束せず scope を先に返す

## 使いどころ

- closed 後の再相談
- `また似た不具合が出た`
- `返金してほしいと言われている`
- `前回の範囲で無料で見てもらえるか`

## 返信文

```text
返金のご希望、確認しました。
トークルームは閉じているため、この場で返金可否やキャンセル手続きを進めるとは断定できません。まずメッセージ上で、前回の修正と今回の再発がつながっているか確認します。
その結果、前回の補足で済むか、新しい依頼として見積り提案が必要かを先にお返しします。いま出ているエラー内容、ログ、または画面スクショを送ってください。
```

## この型で守っていること

- 不安を1文で受ける
- 無料保証を即答しない
- `前回範囲か新規か` を先に返す
- ココナラ取引の返金と、依頼者サービス側の Stripe 顧客返金を混同しない
- closed 後に旧トークルーム継続や承諾取消を前提にしない

# Gold Reply 21 — bugfix と handoff が混ざる相談で、buyer に選択を返さず bugfix へ寄せる

## 使いどころ

- prequote
- `修正からか整理からか判断できない`
- AI/外注継ぎ足しでコードが分からない
- でも今エラーも出ている

## 返信文

```text
ご相談ありがとうございます。判断しづらい状態になっているのは無理もありません。
今出ている不具合を止めるのが先なら、まずは不具合修正から入るのがよさそうです。コード全体の整理は、その結果として流れを把握しておいた方がいいと分かった時点で別に考えられます。
まず困っている症状が分かる画面か文言を送ってもらえますか。その前提で、どちらから入るのが近いかを返します。
```

## この型で守っていること

- `迷っています` を1文で受ける
- buyer に優先順位判断を返さない
- いま止めるべき症状があるなら bugfix を先に置く
- handoff を否定せず、後段の整理として位置づける

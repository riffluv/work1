# Gold Reply 14: quote_sent refuse zoom with anxiety

## 使う場面
- `quote_sent` で Zoom や口頭説明を希望される
- buyer が `文章で伝えるのが苦手` と不安も出している
- 通話不可を伝えつつ、そのまま前に進めたい

## 返信例

```text
ご連絡ありがとうございます。
Zoomでの口頭説明は受けていません。このトークルームで進めます。
文章で伝えるのが苦手でも大丈夫です。こちらで確認したい点を順番に絞って伺います。
```

## 良い理由
- `できません` で終わらず、代替の進め方をそのまま示せる
- buyer の不安を1文で受けてから進められる
- `quote_sent` でも prequote の generic ask に戻らず、今の段階に合う温度で閉じられる

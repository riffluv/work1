# Gold Reply 19 — 成功率不安に、保証せず価値を返す

## 使いどころ

- prequote
- `どれくらいの確率で直るか`
- `15,000円払って分かりませんでしただと怖い`
- 成功率や保証を聞かれているが、冷たく返したくない

## 返信文

```text
ご相談ありがとうございます。
15,000円には原因の切り分けと修正可否の確認を含んでいるので、今ある症状からどこまで進められるかは返せます。割合を固定でお伝えできるものではありませんし、必ず直るともお約束していません。
今困っている症状があれば、その内容をもとにもう少し具体的にお伝えできます。
```

## この型で守っていること

- 成功率を数字で言い切らない
- `分かりませんでしたら怖い` への不安に、15,000円で返る価値を先に示す
- 否定を先頭に置かず、価値説明を先に置く
- prequote の段階なので、重い資料要求に進まない

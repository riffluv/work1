# ココナラ サービス設計（現行）

更新日: 2026-02-13

## 結論（現行の固定モデル）

プラン3段ではなく、以下の「基本 + オプション」で運用する。

1. 基本料金: 15,000円  
内容: 不具合1件の原因切り分けから修正差分の作成（または適用手順の提示）まで

2. 有料オプションA: 追加修正1件（不具合1件） 15,000円

3. 有料オプションB: 追加調査（30分） 5,000円

## 不具合1件の定義

- 同一原因に紐づく1つの現象（1フロー / 1エンドポイント）を1件とする
- 原因または対象フローが別の場合は追加修正として扱う

## 前提

- テキストのみ（通話なし）
- 納品は診断レポート + 修正差分または適用手順
- 本番デプロイ、直接push、新機能開発は標準範囲外
- 秘密情報は値を受け取らず、キー名とマスク情報のみ受け取る
- 最終の実機確認は依頼者環境で実施し、結果ログを受けて追加調整する

## 追加料金の扱い

- 取引中の追加対応: 有料オプション追加で対応
- 正式納品後〜クローズ前: おひねりで追加対応
- クローズ後: 新規依頼で対応

## 運用メモ

- 見積り受付: ON
- 見積り必須: OFF（即購入も許可）
- 同時対応件数: 1件で固定し、品質優先で回す

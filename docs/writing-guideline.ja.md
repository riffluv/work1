# 文章ガイドライン（ココナラ取引メッセージ用）

更新日: 2026-02-12
根拠資料: `docs/文章コミュニケーション完全ガイド.md`

このファイルは、ココナラでのお客さんとのやりとり文を書くときのルール。
Codex（下書き）もClaude Code（リライト）も、このガイドラインに従うこと。

---

## 基本方針

- **です・ます調**がベース。尊敬語・謙譲語は必要な箇所だけ
- **メールの丁寧さ + チャットのテンポ**（堅すぎず、雑すぎず）
- トーンの目安: 「フリーランスのエンジニアが書いたメッセージ」
- 文章の目的: 礼儀の誇示ではなく、**誤解防止と合意形成**
- **1メッセージ1目的**を基本にする。相談、報告、依頼を混ぜすぎない

## 毎回守る構成（この順番で書く）

1. **要点**（結論 / 現状）
2. **根拠**（ログ / 再現 / 観測）
3. **次の作業**（こちらが何をするか）
4. **相手にしてほしいこと**（返答の形式 + 期限）

結びは、できるだけ相手の次の行動が分かる形で終える。

## 敬語のルール

| ルール | 理由 |
|---|---|
| 「させていただく」は0〜1回まで | 連発するとAI臭い。許可・恩恵の文脈以外では「します」「いたします」 |
| 二重敬語を避ける | 「おっしゃられる」→「おっしゃる」 |
| 過剰な謙譲は使わない | 「幸甚に存じます」「ご高配を賜り」はココナラでは距離が開きすぎる |
| 結びを毎回変える | 「何卒よろしくお願いいたします」の連続は機械感が出る |

## NG表現 → OK表現（置換リスト）

| NG（AI臭い） | OK（自然） |
|---|---|
| 鋭意対応させていただきます | 確認します。○時までに一次報告します |
| 善処いたします | Aを確認し、必要ならBを修正します |
| ご査収ください | 添付をご確認ください |
| 幸甚に存じます | 助かります / お願いします |
| 差し支えなければご教示賜り… | 可能なら○○を教えてください |
| 念のため申し添えます | 一点だけ補足です |
| 諸般の事情を総合的に鑑み | （削除。具体的に書く） |
| ご確認いただけますと幸いです（毎回） | 問題なければ「OK」と一言ください |
| 追ってご連絡申し上げます | ○時までに連絡します |

## AI臭さを消す5つのルール

1. **固有情報を3つ以上入れる** — 日時、対象、次のアクション（「鋭意対応」だけで終わらない）
2. **「させていただく」を0〜1回に抑える** — 「します」「いたします」で十分
3. **文の長さを混ぜる** — 短文→中文→短文のリズム。均一な長さは機械的に見える
4. **接続詞の固定を避ける** — 「なお、また、さらに」の連発は箇条書きで代替
5. **相手の次の行動を型で渡す** — 「OKか修正点を一言ください」のように選択肢を出す
6. **相手の言葉を一度拾ってから質問する** — いきなり質問票を出すと bot 感が出る
7. **抽象的な安心語より、具体的な事実を書く** — 「ご安心ください」より「勝手に料金が増えることはありません」

## ココナラ禁止事項（絶対守る）

| 禁止 | 代替 |
|---|---|
| 外部連絡に誘導（LINE, Discord, Slack, Zoom, メール） | トークルーム内で完結 |
| 外部共有に誘導（GitHub, Google Drive, 外部URL） | トークルーム内の添付機能で共有 |
| 外部決済（銀行振込, PayPay等） | ココナラの決済のみ |
| 個人情報の不要な要求 | 必要最低限のみ |
| 未完成で「正式な納品」を押す | 完成物がある場合のみ |
| トークルーム内容の外部公開 | しない |

## 速度のルール

| 場面 | 目標 |
|---|---|
| 初回返信（購入直後） | 30分〜2時間以内。一言でもいいから送る |
| 追加質問 | 調査中に判明した時点ですぐ |
| 進捗報告 | 1日1回 or 重要な節目 |
| トラブル報告 | 判明した瞬間に即 |

**48時間ルール**: 購入後48時間以内に一度もメッセージしないと自動キャンセル＋★1。

## 評価を守るルール

- 評価コメントへの返信は「感謝 + 改善意欲」だけ。不満は絶対に書かない
- 「星5だと思っていました」は見込み客にも見えるので絶対NG
- 評価項目に「コミュニケーション（スムーズさ・マナー）」があるので、文章の質 = 評価

## リピートにつなげる一言

納品完了時に以下を添える:
> 同様の不具合が再発したら、このトークルームに続けて状況を書いてください（クローズ後はメッセージ機能でOKです）。

## 進捗報告テンプレ（「調査中」で終わらせない）

```
進捗:
分かったこと:
残っている不明点:
次にやること（期限）:
お願い（返答形式＋期限）:
```

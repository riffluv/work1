## 要点（実運用で重要な事実）
- 見積り相談には4経路ある。  
サービス経由 / プロフィール経由 / メッセージ経由 / トークルーム経由。

- トークルーム経由は「定期購入のみ」。  
単発はメッセージ経由で提案する。

- 見積り提案の購入期限は「最終提案更新日から1週間」。  
期限切れは再提案でリセットされる。

- 提案文に絵文字があるとエラーになる場合がある。  
提案本文は絵文字を入れない。

- プロフィール経由・メッセージ経由は、放置するとサービス実績に紐付かない。  
提案時に「出品サービスに紐付ける」を必ず使う。

- サービス紐付けは提案時のみ可能。  
購入後の後付けはできない（原則不可）。

- 紐付け可能な対象サービスは「公開中 / 受付休止中 / 満枠対応中」（電話相談除く）。

- 依頼経路（サービス経由かプロフィール経由か）は見積り詳細で確認できる。

- サービス経由の見積り受付は出品編集からON/OFFできる。  
「購入前見積り必須」は売れやすさ低下の可能性があるため、必要なサービスのみONが推奨。

- プロフィール経由の見積り受付はプロフィール設定でON/OFFできる。

## このworkspaceの固定運用
- サービス経由見積り受付: ON
- 購入前見積り必須: OFF
- プロフィール経由見積り受付: ON
- プロフィール経由 / メッセージ経由の提案時は、毎回「出品サービスに紐付ける」

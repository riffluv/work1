# ココナラ返信 Ban リスト

目的:
- Codex が出しがちな機械文・AI文を事前に弾く
- Claude 監査で繰り返し問題になった表現を再発させない

使い方:
- 返信文を作る前に、この一覧に引っかかる表現を避ける
- 書いた後にも再確認する
- 迷ったら「短く、直接、事実ベース」に戻す

## 強禁止
- `返金` と `承知しました` の組み合わせ
- `お気持ちももっともです`
- `もっともです`
- `ように見えた`
- `返金についてすぐに断定する前に`
- `ご想像`
- `認識のずれ`
- `確認案件`
- `整理してお返しします`
- `作業として発生します`
- `別件として整理する`
- `スコープを切って`
- `形になります`
- `切り分けて考える`

## 基本禁止
- `安全です`
- `自然です`
- `合っています`
- `分かりやすいです`
- `見た目は軽く見えても`
- `気になると思いますので`
- `必要情報がそろい次第`
- `ご共有ください`
- `ご査収ください`
- `対応範囲外です。` だけで終わる書き方
- `対応していません。` だけで終わる書き方
- `させていただきます` の連発
- `〜自体は`

## 条件付き禁止
- `承知しました`
  - 苦情 / 再発 / 返金 / 金額不満では使わない
  - 通常の受領や軽い受け答えなら可
- `新しい相談`
  - closed 後の初回返信では前面に出さない
- `規約上は`
  - 苦情や不満が強い相手への初回返信では使わない
- `保証はしていません`
  - 感情注意ケースの初回返信では使わない

## 再発しやすい型
- `本来はどの〜想定でしょうか`
- `まず状況を確認させてください`
- `確認して、2時間以内には見立てをお返しします`
- `〜の件も含めて、まず確認させてください`

上記は全面禁止ではないが、連続使用でテンプレ感が出るため、同じバッチ内で繰り返さない。

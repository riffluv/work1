# internal-quality-samples

この階層は、**実務で毎回参照する内部品質本丸**です。

- `bugfix-15000/`
  - 不具合修正サービスの内部品質見本
  - `00_結論と確認方法.md` の温度感
  - 修正済みファイルのコメントの濃さ
  - patch の見せ方
- `handoff-25000/`
  - 引き継ぎ整理サービスの内部品質見本
  - `00_結論と要点.md` と `01_[対象フロー名]_引き継ぎメモ.md` の温度感
  - 危険箇所 / 次の着手順の粒度

外向けの購入前見本は別です。

- `/home/hr-hm/Project/work/サービスページ/購入前見本/`
- `/home/hr-hm/Project/work/サービスページ/送付文/`

使い分け:
- `internal-quality-samples`
  - 実案件の納品物を作る時に参照する本丸
- `サービスページ/購入前見本`
  - 見積もり相談時に購入者へ見せる見本
- `サービスページ/送付文`
  - 見積もり相談や正式納品で、相手にそのまま貼る外向け文面の本丸

題材:
- 現在の両サービスの本丸サンプルは、公開リポジトリ `nextjs/saas-starter` の Stripe 決済フローを題材にしています。

# ココナラ返信 Self-Check

目的:
- 本番で Claude を毎回使わなくても、Codex が最低限の日本語品質チェックをできるようにする

## 送信前チェック
- 相手の質問に直接答えているか
- 相手がすでに書いた情報を聞き直していないか
- 見立て・確認観点を1文入れたか
- 質問は本当に1〜2点で足りるか
- 時刻目安を入れたか
- 直近3件と同じ書き出し・末尾になっていないか

## Ban チェック
- `整理してお返しします` が残っていないか
- `もっともです` が残っていないか
- `ように見えた` が残っていないか
- `確認案件` が外向けに残っていないか
- `返金` と `承知しました` が近接していないか
- `安全です / 自然です / 合っています / 分かりやすいです` の判定形容詞が残っていないか
- `別件として整理する` `スコープを切って` など内部語が残っていないか

## ケース別チェック

### 見積り初回
- 質問前に見立て1文を入れたか
- `本来はどの〜想定でしょうか` の固定構文に戻っていないか

### 感情注意
- 不便の認知があるか
- 感情の要約承認になっていないか
- 最初の依頼が重すぎないか
- 金額に触れた相手なら、その論点を無視していないか

### quote_sent
- 聞かれたことに結論から答えているか
- ココナラ仕様の質問で `条件による` だけで終わっていないか

### closed 後再発
- 確認依頼文が毎回同じになっていないか
- 自責寄りなら、短い安心提供があるか
- 金額不安なら、方向性だけは触れているか

### 範囲外
- `別件で受けられる` と `現在は受けない` を混同していないか
- 次の行動を1文で示しているか

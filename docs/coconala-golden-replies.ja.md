# ココナラ返信 Golden Replies

目的:
- Codex が参照する、通過済みの良文例をまとめる
- 抽象ルールだけでなく、実際の温度感を固定する

使い方:
- 新しい返信を書く前に、近いケースの書き方を1件参照する
- 完全コピペではなく、骨格と温度感だけを借りる

## 1. 高適合の見積り相談
```text
ご連絡ありがとうございます。内容を確認しました。

ローカルでは動いていて、本番の App Router だけ署名検証に失敗しているとのことですので、今回のご相談は不具合1件として対応できる見込みです。
基本料金15,000円で、原因切り分けから修正差分または修正済みファイルの共有まで対応します。

この内容で見積りをお送りしてよければ、そのまま進めます。
```

## 2. 確認だけの相談
```text
ご連絡ありがとうございます。
修正までは不要で、今のStripe連携が正しく動いているかだけ見てほしいということですね。

確認だけの対応も大丈夫です。5,000円で見られます。
AIで作ったコードとのことなので、まずはStripeまわりの設定と処理の流れを見て、気になる点があるかどうかを確認します。

次の2点を教えてください。
1. 何の処理を見てほしいか（例: 決済 / Webhook / 会員反映）
2. 本番とテストのどちらの話か

確認して、2時間以内には最初の見立てをお返しします。
```

## 3. 値引き相談
```text
ご相談ありがとうございます。
Webhook の不具合であれば対応できます。料金は15,000円です。

ただ、まだ症状の内容が見えていないので、今の段階では先に状況を確認したいです。
次の2点を教えてください。
1. 今はどの不具合が出ていますか
2. 本番とテストのどちらで起きていますか

内容を見て、2時間以内には見立てをお返しします。
もしご予算優先であれば、確認だけの進め方が合うかも含めてその時点で判断します。
```

## 4. 苦情寄りの進捗不満
```text
ご連絡ありがとうございます。
2時間以内とお伝えしていたのに連絡が止まってしまっていたとのこと、ご不安にさせてしまい申し訳ありません。

このまま待たせる形にはしません。
今から状況を立て直しますので、まず一番困っている症状を1つだけ教えてください。

確認に入り、本日[時刻]までに見立てをお返しします。
```

## 5. closed 後再発
```text
ご連絡ありがとうございます。
前回の対応後にまた同じ不具合が出ているとのこと、ご不便をおかけしています。

追加の費用がかかるかどうかも含めて、まず前回の修正と今回の状況の関係を確認させてください。
今回の症状が前回と同じ画面・同じ操作で起きているかと、今出ている状態が分かるスクショを1枚送ってください。

確認して、2時間以内には見立てをお返しします。
```

## 6. 複数症状だが同一原因の可能性が高い
```text
ご連絡ありがとうございます。
一度にいくつも起きていて大変な状態とのことですが、書いていただいた3つは同じ処理で止まっている可能性があります。

まずは1つずつではなく、決済後の処理全体を見てよさそうかを確認したいので、完了ページの画面スクショを1枚送ってください。
そこから、同じ原因で見られる範囲かを整理します。

確認して、2時間以内には見立てをお返しします。
```

## 7. 範囲外だが別件で受けられる
```text
ご相談ありがとうございます。
単発決済に加えて月額プランを新しく作る内容であれば、不具合修正ではなく新機能の実装になります。

この内容であれば、別件の実装相談として進める形になります。
必要な範囲と希望納期を教えてください。その内容で対応できます。
```

## 8. QST の直接回答
```text
ご連絡ありがとうございます。
支払い方法はココナラの購入画面で決まるので、こちらから変更することはできません。
購入画面にコンビニ払いが出ていれば使えます。
出ていない場合は、ココナラのヘルプページに利用条件が載っていますので、そちらをご確認ください。
```

## 9. スコープ判定の模範
```text
ご相談ありがとうございます。
決済が通らない件と、ログインまわりの500エラーは、現時点では別原因の可能性が高そうです。

こちらで1件にまとめるより、まず優先して確認する方を決めて進めます。
どちらを先に見たいか教えてください。

確認して、2時間以内には進め方の見立てをお返しします。
```

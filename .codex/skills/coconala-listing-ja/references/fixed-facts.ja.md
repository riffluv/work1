# 固定条件（変更禁止）

このファイルの項目は、ユーザーから明示的な変更指示がない限り変更しない。

## 価格・提供モデル
- 基本料金: 15,000円
- 有料オプション: 追加修正1件 15,000円
- 15,000円は調査だけで終わる形ではなく、原因確認から修正・確認手順・修正済みファイル返却まで進める前提で案内する

## 不具合1件の定義
- 同一原因に紐づく不具合1件
- 対象は1フローまたは1エンドポイント
- 原因または対象フローが別なら追加対応

## 標準範囲外
- 通話・ビデオ会議
- 依頼者リポジトリへの直接 push
- 本番デプロイ代行
- 新機能開発・仕様追加

## 運用上の固定
- やり取りはすべてメッセージ（文字）で行う
- 通話・ビデオ会議は対応しない
- 最終確認は依頼者環境で実施
- 確認結果を受けて追加調整
- 初回返信は24時間以内（通常数時間以内）
- 必要情報が揃い次第、48時間以内を目標に切り分け結果と修正方針を案内する
- 見積り相談経由の取引では有料オプションの追加購入はできない
- 正式納品後の追加費用は、クローズ前ならおひねりで案内する
- クローズ後の追加対応は新規依頼で案内する

## セキュリティ・規約
- `.env` の値やAPIキー値は要求しない
- 外部連絡誘導をしない
- 外部共有誘導をしない
- 外部決済誘導をしない

# 置換辞書（汎用）

## 方針
- 過剰敬語・冗長表現を短く自然にする。
- 意味を変えずに言い換える。

| NG表現 | 推奨表現 |
|---|---|
| ご確認のほどよろしくお願いいたします | ご確認ください |
| ご確認いただけますでしょうか | ご確認いただけますか |
| ご教示いただけますと幸いです | 教えてください |
| ご共有いただけますと幸いです | 共有いただけると助かります |
| 確認させていただきます | 確認します |
| 対応させていただきます | 対応します |
| 共有させていただきます | 共有します |
| 〜かと存じます | 〜と思います |
| 〜ではないでしょうか | 〜の可能性があります |
| こちらが回答になります | こちらが回答です |
| 〜のほう | 〜を |
| 〜という形で | 〜として |
| 追ってご連絡申し上げます | ○時までに連絡します |
| 善処いたします | Aを確認し、必要ならBを修正します |
| 諸々確認いたします | ○と○を確認します |
| 基本的には | 条件があれば明記し、なければ断定 |
| なるべく早く | ○日○時までに |
| ご査収ください | 添付をご確認ください |
| 何卒よろしくお願い申し上げます | よろしくお願いします |
| ご不明点がございましたら | 不明点があればお知らせください |
| お忙しいところ恐縮ですが | お手すきのときに |
| 大変恐縮ではございますが | 1点だけ確認させてください |
| 〜していただければと思います | 〜をお願いします |
| 了解しました | 承知しました（チャットでは「了解です」も可） |
| 可能でしょうか | できますか |
| 問題ございません | 問題ありません |
| 取り急ぎ | まずは |
| そのような認識で相違ないでしょうか | その認識で合っていますか |
| 〇〇の件につきまして | 〇〇について |
| ご連絡差し上げます | 連絡します |

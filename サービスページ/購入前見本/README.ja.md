# 購入前見本

この階層は、購入前に相手へ見せる**見本ファイル本体**の置き場です。

- `bugfix-15000/`
  - 不具合修正サービスの見本
- `handoff-25000/`
  - 整理・引き継ぎサービスの見本

使い分け:
- 相手にそのまま貼る案内文は `/home/hr-hm/Project/work/サービスページ/送付文/購入前見本/`
- 実務の内部品質基準は `/home/hr-hm/Project/work/docs/internal-quality-samples/`

補足:
- ここに置くのは、購入前に「どんな形で返すか」をイメージしてもらうための見本です
- 実際の納品では、案件内容に合わせて対象・原因・確認方法・危険箇所の具体名が差し替わります

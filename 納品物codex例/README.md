# 納品物codex例

2026-03-30 時点の Internal OS を前提にした、納品物の見本です。

- `bugfix-15000_public`: 現在 public の live サービス。購入者向け納品物と、修正済みZIP想定の中身例を置いています。
- `handoff-25000_private`: 現在は private の内部準備用サービス。購入者向け納品物の見本だけを置いています。

共通ルール:
- `購入者向け` には、実際に渡す前提のファイルだけを置く
- `内部確認用` には、Scope Snapshot や正式納品メッセージ案など、購入者へ渡さない確認用メモだけを置く
- `handoff-25000` は未公開のため、外向け案内や購入導線には使わない

このフォルダの目的は、現行OSで「どう納品されるか」をサービス別に見える化することです。
